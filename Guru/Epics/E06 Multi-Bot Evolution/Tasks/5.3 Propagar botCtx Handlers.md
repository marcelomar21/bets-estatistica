# 5.3 Propagar botCtx para todos os handlers

## Objetivo

Cada handler recebe `botCtx` como parametro, substituindo todas as leituras de `config.telegram.*` por `botCtx.*`.

## Action

1. Cada handler recebe `botCtx` como parametro
2. Substituir todas as leituras de `config.telegram.*` por `botCtx.*`
3. Substituir chamadas a `sendToAdmin(text)` por `sendToAdmin(text, botCtx)`

Handlers a modificar:
- `bot/handlers/adminGroup.js` — Router de mensagens admin
- `bot/handlers/admin/actionCommands.js` — Comandos admin (`/postar`, `/odds`, etc)
- `bot/handlers/callbackHandlers.js` — Callbacks leem `config.telegram.publicGroupId` (linha 73)
- `bot/handlers/startCommand.js` — `/start` le `config.telegram.publicGroupId` (linhas 204, 285, 528)
